# Tailor Management System (TMS) - Elite Edition

A premium, modern, and responsive web application designed for high-end tailoring businesses. This system is built using pure **HTML5, CSS3, and Vanilla JavaScript**, requiring no external frameworks or backend.

## ✨ Features

- **Luxury UI/UX**: Professional Black & Gold theme with glassmorphism, smooth animations, and a responsive dashboard.
- **Customer CRM**: Complete management of customer profiles, contact details, and history.
- **Precision Measurements**: Specialized body measurement tracking for bespoke tailoring.
- **Order Lifecycle**: Track orders from 'Pending' through 'Cutting', 'Stitching', and 'Ready' to 'Delivered'.
- **Smart Analytics**: Real-time stats and status overview using the HTML5 Canvas API.
- **Professional Invoicing**: Generate and print elegant, high-quality invoices for customers.
- **Data Persistence**: All data is securely stored in the browser's Local Storage.
- **Dark/Light Mode**: User preference support with theme toggling.

## 🚀 Getting Started

## 📂 Project Structure

- `index.html`: Login gateway with fake authentication.
- `dashboard.html`: Main analytics and overview.
- `customers.html`: Customer CRUD and search.
- `measurements.html`: Detailed body measurement forms.
- `orders.html`: Order tracking and management.
- `invoices.html`: Printable invoice generator.
- `settings.html`: Shop configuration and data management.
- `js/`: Modular JavaScript logic for storage, UI, and page-specific features.
- `css/`: Themed styles including responsive design and custom animations.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser LocalStorage API
- **Charts**: HTML5 Canvas API
- **Icons**: Font Awesome 6.4
